# cauchy-haskell

Haskell packages of the *cauchy* family:

- **monoid-semiring** — finitely supported functions from a monoid to a
  semiring, multiplied by convolution (the generalized Cauchy product).
- **cauchy-poly** — univariate polynomials `S[x]` and lazy power series
  `S[[x]]` built on `monoid-semiring`.
- **cauchy-oracle** — differential-testing harness against external
  referees (Singular, Sage, numpy, in-library engines).

This repository is a **read-only publish target**: snapshots of the
`haskell/` subtree of a private development repository, pushed at release
time. Issues and bug reports are welcome here; pull requests cannot be
merged directly but patches posted in an issue will be applied upstream.
